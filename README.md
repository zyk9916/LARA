# LARA
论文复现《LARA: Attribute-to-feature Adversarial Learning for New-item Recommendation》

学号：2020223045112

姓名：周煜坤

数据采用MovieLens 1M

main.py：主函数，启动整个程序。

load_data.py：对数据进行预处理并读取数据。

model.py：定义超参数、生成器/判别器，训练函数和启动函数。

test.py：对模型进行评估，计算ndcg，也可输出similar_user的相关信息。
